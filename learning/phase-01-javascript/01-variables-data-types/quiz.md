# Part 1 – Multiple Choice Questions (30)

---

## Q1

Which keyword is recommended by modern JavaScript?

A. var

B. let

C. const

D. static

✅ Answer: C

Explanation:

Always use **const** unless reassignment is required.

---

## Q2

Which keyword allows redeclaration?

A. let

B. const

C. var

D. None

✅ Answer: C

---

## Q3

C. const

D. object

✅ Answer: C

## Q4

# Module 01 - Exercise Solutions

Welcome to the solutions for **Module 01 – Variables & Data Types**.

## Before You Continue

These solutions are intended to help you verify your work and understand best practices.

Try to complete every exercise on your own before viewing the solution.

---

## Best Practices Used

- Meaningful variable names
- `const` by default
- `let` only when reassignment is required
- Proper code formatting
- Comments where appropriate
- Template literals instead of string concatenation

---

```bash
node solution-01.js
```

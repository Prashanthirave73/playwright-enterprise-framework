# Challenge 20 ⭐⭐⭐⭐⭐

## Mini Playwright Framework Configuration

Create an object containing:

```javascript
const framework = {
  browser: 'Chromium',
  baseURL: 'https://example.com',
  timeout: 30000,
  retries: 2,
  reporter: 'HTML',
  headless: true,
};
```

Print the configuration exactly as a real framework startup log.

Example:

```
========== FRAMEWORK CONFIGURATION ==========

Browser : Chromium

Base URL : https://example.com

Timeout : 30000

Retries : 2

Reporter : HTML

Headless : true

============================================
```

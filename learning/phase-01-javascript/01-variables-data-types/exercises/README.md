# Module 01 - Exercises

Welcome to the practice section for **Variables & Data Types**.

## Objective

The purpose of these exercises is to strengthen your understanding of:

- Variables
- Data Types
- Arrays
- Objects
- Template Literals
- typeof
- Type Conversion

---

## Instructions

- Complete each exercise without referring to the solutions.
- Execute each file using Node.js.
- Compare your output with the expected output.
- Follow JavaScript best practices.
- Use `const` whenever possible.
- Use `let` only when reassignment is required.
